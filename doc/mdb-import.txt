NAME
  mdb-import - Import CSV data into an MDB database table.

SYNOPSIS
  mdb-import [options] database table csvfile

DESCRIPTION
  mdb-import is a utility program distributed with MDB Tools. 

  It reads a csvfile and insert the data into the given table of a mdb database file.

OPTIONS
  -H row              Skip row header rows.
  -d delimiter        Specify a column delimiter.

NOTES
  Only a limited set of column types are supported:
  * Text
  * Byte
  * Int
  * Long Int
  * Bool

ENVIRONMENT
  MDB_JET3_CHARSET    Defines the charset of the target JET3 (access 97) file. Default is CP1252. See iconv(1).
  MDBICONV            Defines the output charset to use for the SQL file. Default is UTF-8. mdbtools must have been compiled with iconv.
  MDBOPTS             semi-column separated list of options:
                      * use_index
                      * no_memo
                      * debug_like
                      * debug_write
                      * debug_usage
                      * debug_ole
                      * debug_row
                      * debug_props
                      * debug_all is a shortcut for all debug_* options

HISTORY
  mdb-import first appeared in MDB Tools 0.6.

AUTHORS
  The mdb-import utility was written by Brian Bruns.

BUGS
  Referential integrity is not enforced.

  Encrypted files are not supported.
